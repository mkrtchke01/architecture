# LUMÉRA — лендинг архитектурного декора, света и стекла

Премиальный продающий лендинг для компании, которая проектирует, производит и монтирует
индивидуальные архитектурные, интерьерные и ландшафтные декоративные решения с интегрированным
освещением, а также светопрозрачные конструкции (козырьки, перегородки, зенитные фонари).

**Стек:** React 18 + TypeScript + Vite + Tailwind CSS.
Без сторонних UI-библиотек: иконки — inline SVG, анимации — IntersectionObserver + CSS.

---

## 1. Концепция

«Архитектурный свет в темноте». Тёмная luxury-эстетика, много воздуха, крупная display-типографика
(serif Cormorant + sans Manrope), латунный (brass) акцент, мотивы стекла (blur/прозрачность),
световых бликов и архитектурной сетки. Сдержанные анимации: плавное появление, hover-состояния,
мягкие переходы. Визуально — бренд из мира архитектуры и high-end construction, без шаблонного
«строительного» дизайна и кричащих цветов.

Позиционирование на двух аудиторий одновременно:
- **B2B** (архитекторы, дизайнеры, застройщики) — «надёжный производственный партнёр»;
- **частные клиенты** — «поможем разобраться и доведём проект до результата».

## 2. Структура лендинга

1. **Header** — фиксированный, якорная навигация, телефон, CTA, мобильное меню.
2. **Hero** — заголовок, подзаголовок, 2 CTA, доверительный маркер, метрики.
3. **Services** — «Что мы создаём» (9 направлений).
4. **B2BSection** — «Для архитекторов и бюро» (7 пунктов сотрудничества).
5. **PrivateClientsSection** — польза для частных клиентов простым языком.
6. **Solutions** — карточки-направления с изображениями.
7. **Portfolio** — проекты с фильтрами по типу объекта (задача/решение/материалы + CTA).
8. **Process** — 8 этапов работы.
9. **Advantages** — 8 преимуществ.
10. **Trust** — маркеры доверия, отзывы, логотипы партнёров.
11. **FAQ** — доступный аккордеон (9 вопросов).
12. **CTA** — финальный призыв.
13. **ContactForm** — форма заявки (имя, телефон, email, тип клиента, тип задачи, описание,
    файл, способ связи, согласие) — frontend-заглушка, готовая к API.
14. **Footer** — бренд, навигация, контакты, реквизиты.
15. **FloatingContact** — плавающая кнопка мессенджера.

## 3. Тексты

Все тексты — на русском, в премиальном экспертном тоне, без канцелярита и дешёвых обещаний.
Хранятся в двух местах:
- `src/data/content.ts` — услуги, решения, портфолио, этапы, преимущества, FAQ, отзывы;
- `src/data/site.ts` — бренд, контакты, навигация, метрики;
- остальной копирайт (заголовки секций, B2B/частным клиентам, CTA, форма) — внутри компонентов.

## 4. UI/UX-рекомендации

- CTA повторяются в Header, Hero, B2B, Private, Solutions, Portfolio, CTA, Form и плавающей кнопке.
- Навигация простая и якорная: Услуги · Решения · Портфолио · Этапы · Для архитекторов · Контакты.
- Адаптив: desktop / tablet / mobile (Tailwind breakpoints), мобильное меню с блокировкой скролла.
- Доступность: skip-link, видимый `:focus-visible`, `aria-*` на интерактиве, `alt` у изображений,
  `aria-live` для статуса формы, поддержка `prefers-reduced-motion`, контрастная палитра.
- Производительность: hero-изображение `fetchPriority="high"`, остальные `loading="lazy"`.
- Для реальных фото держите единый стиль (тёмные, с акцентом на свет/стекло) — это сохранит премиальность.

## 5. Запуск проекта

Требуется Node.js 18+.

```bash
npm install      # установка зависимостей
npm run dev      # запуск dev-сервера (http://localhost:5173)
npm run build    # production-сборка в папку dist/
npm run preview  # локальный просмотр production-сборки
```

Деплой: статическая сборка `dist/` — подойдёт Vercel, Netlify, Cloudflare Pages, GitHub Pages
или любой статический хостинг / nginx.

## 6. Где заменить заглушки на реальные данные

**Контакты и бренд → `src/data/site.ts`:**
- название бренда, юр. наименование, год основания;
- телефон (`phone` + `phoneHref`), email (`email` + `emailHref`);
- город, адрес, режим работы;
- ссылки на мессенджеры и соцсети: `whatsapp`, `telegram`, `instagram`;
- метрики в `stats` (годы, кол-во проектов, гарантия).

**SEO и микроразметка → `index.html`:**
- `<title>`, `<meta name="description">`, canonical, Open Graph (`og:url`, `og:image`);
- JSON-LD `LocalBusiness`: name, телефон, адрес, координаты, `sameAs` (соцсети);
- превью-картинка `og-cover.jpg` (положить в `public/`).

**Контент → `src/data/content.ts`:**
- изображения услуг/решений/портфолио (поля `image`) — заменить Unsplash на реальные фото;
- тексты проектов в `projects` (задача / решение / материалы);
- отзывы — в `src/components/Trust.tsx` (`testimonials`): цитаты, имена, роли;
- логотипы партнёров — `partnerLogos` (текст → реальные логотипы SVG/PNG).

**Форма → `src/components/ContactForm.tsx`:**
- подключить отправку заявки (свой backend `/api/lead` или сервис Formspree/Getform) — см. TODO;
- ссылка на политику конфиденциальности.

**Документы → `src/components/Footer.tsx`:**
- ссылки на политику конфиденциальности и оферту.

> Все места помечены комментарием `// TODO:` — найдите их поиском по проекту.
